Vidra Video Project

Build Video IR

Run npm run build:video to build the SDK project in src/video.js into Vidra JSON IR.

Render to MP4

Run npx @sansavision/vidra render video.vidra -o output.mp4 to render the VidraScript project.

Visual Editor

Run npx @sansavision/vidra editor video.vidra to open the browser-based visual editor.

Web Scene

The web/chart.html file uses @sansavision/vidra-web-capture to sync with the video timeline.
